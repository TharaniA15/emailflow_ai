# EmailFlow AI

EmailFlow AI is a professional email classification and response assistant. This frontend UI provides a clean and easy interface for managing email insights, summaries, and intelligent actions.

## Available Scripts

In the project directory, you can run:

### `npm start`

Runs the app in development mode.
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page reloads automatically when you make changes.
Lint errors appear in the console.

### `npm test`

Launches the test runner in interactive watch mode.

### `npm run build`

Builds the app for production into the `build` folder.
The build is minified and filenames include hashes for cache optimization.

### `npm run eject`

**Note: this is a one-way operation. Once you `eject`, you can't go back!**

If you choose to eject, this will copy the build configuration and dependencies into your project so you can customize them directly.
